# UI 布局修正与重构计划

根据您的反馈，我将废弃目前的左侧 Dock 栏设计，改为**顶部导航栏 + 左侧动态内容区**的布局。

## 1. 布局重构 (`app/ui/main_window.py`)

### A. 顶部主导航栏 (Main Navigation Bar)
*   **位置**: 窗口最顶部（原 ToolBar 位置，替换或重构之）。
*   **内容**: 水平排列的功能入口按钮。
    *   **"JSON工具"**: 选中时，左侧区域显示 JSON 相关功能面板。
    *   **"文本格式化"**: 选中时，左侧区域显示文本清洗规则面板。
    *   (保留) **"批处理" / "导出" / "预设" / "主题"**: 作为辅助功能按钮放在右侧或次级位置。
*   **交互**: 点击导航按钮切换左侧内容区 (`QStackedWidget`) 的页面。

### B. 左侧动态内容区 (Left Sidebar)
*   **位置**: 窗口左侧，宽度固定（约 240-280px）。
*   **内容 (`QStackedWidget`)**:
    *   **页面 1 (JSON工具)**: 包含 JSON 校验、美化、压缩、转义等操作按钮及配置。
    *   **页面 2 (文本格式化)**: 包含原有的“去除空格”、“去除分隔符”等复选框规则。
*   **样式**: 去除之前的 Dock 折叠动画，改为直接切换显示内容，保持界面稳定。

### C. 右侧工作区
*   保持现有的 **输入/输出分栏 + 局部 Header** 布局不变。

## 2. 样式更新 (`app/ui/styles.qss`)
*   **顶部导航栏样式**: 设计类似 Tab 或菜单栏的样式，选中项有底部亮条或背景高亮，体现“入口”概念。
*   **移除**: 之前为 Dock 栏设计的深色垂直样式。

## 3. 实现步骤
1.  **移除 Dock 栏**: 删除 `dock_bar` 及其相关布局代码。
2.  **重构顶部工具栏**:
    *   创建一个新的顶部容器 `top_nav_bar`。
    *   添加“JSON工具”和“文本格式化”两个互斥的 `QPushButton` (或 `QToolButton`)。
    *   将原有的辅助功能（批处理、导出等）整合到这个导航栏的右侧，或者保留在下一行的 `QToolBar` 中（建议整合以节省纵向空间，或保持两行：第一行主功能切换，第二行辅助工具）。
    *   **决策**: 为了简洁，我将把“JSON工具”和“文本格式化”做成显著的 **主 Tab 切换**，放在最顶端；原有的 `QToolBar` (批处理/导出等) 保持在第二行或作为 Tab 页面内的局部工具栏。
    *   **修正方案**: 参考 json.cn，顶部是黑底/深色底的导航条。我将实现一个深色顶部导航栏，包含 [JSON工具] [文本格式化] ... [主题]。
3.  **调整左侧 Stack**: 保持 `sidebar_container` 和 `sidebar_stack`，但由顶部导航按钮控制切换。
4.  **更新 QSS**: 适配新的顶部导航栏样式。
